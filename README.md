# 🧮 BMI Calculator Project

A modern and user-friendly web application that allows users to quickly calculate their Body Mass Index (BMI) using height and weight inputs. The application provides instant feedback and categorizes the BMI result into health ranges with a clean and interactive interface.


## 📝 What Does It Do?

The app allows users to calculate and understand their BMI in a simple and intuitive way:

BMI Calculation:
Users enter their height and weight using interactive sliders to instantly calculate their Body Mass Index.

Real-Time Input Display:
Height and weight values update live while the sliders are being adjusted.

Health Category Detection:
After calculation, the app classifies the result into standard BMI categories such as Underweight, Normal, Overweight, or Obese.

Visual BMI Range Guide:
Color-coded labels help users quickly understand where their BMI result falls within common health ranges.

Clean User Interface:
The application provides a minimal and modern design that focuses on clarity and usability.


## ⚙️ How Does It Work?

Vanilla JavaScript Architecture:
The project is built using pure HTML, CSS, and JavaScript without any frameworks, focusing on simple and efficient DOM manipulation.

Dynamic BMI Calculation:
JavaScript calculates BMI using the standard formula:

BMI = weight / (height²)

Interactive Sliders:
Range sliders allow users to adjust height and weight easily, improving usability compared to traditional input fields.

Instant UI Updates:
Once the calculate button is clicked, the result and category are displayed immediately on the page.

Responsive Layout:
The layout is designed with modern CSS techniques to work well on both desktop and mobile devices.
